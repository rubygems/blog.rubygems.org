RubyGems Blog
=============

The official RubyGems blog.

Setup
=====

Make sure you have jekyll installed (`gem install bundler` / `bundle`), and run:

    $ jekyll serve

The pages will be available at [http://localhost:4000/](http://localhost:4000/).
